TECH STACK
═══════════════════════════════════════════════════════════════════════

Framework   : Next.js 14 (App Router)
Language    : JavaScript (no TypeScript)
Styling     : CSS Modules + Global CSS (NO Tailwind, NO Bootstrap)
State       : React useState / useEffect / useContext / useReducer
Icons       : react-icons (fa6, gi, tb, md sets)
Fonts       : next/font/google:
              "Rozha One"        → headings, numbers
              "Noto Sans Bengali" → all Bangla text, body
              "Bebas Neue"       → labels, badges, caps
              "Fira Code"        → IDs, numbers, code values
Data        : JS arrays in /data/ (no database, no API)
Charts      : Pure CSS (no chart libraries)

DESIGN CONCEPT — "WARM WORKSHOP DASHBOARD"
═══════════════════════════════════════════════════════════════════════

Theme: A master craftsman's workshop logbook meets a modern SaaS 
dashboard. Warm parchment backgrounds, rich walnut browns, honey 
gold accents, wood grain textures on every surface. Feels like 
you're managing a premium handcrafted furniture business.

NEVER USE: Black, dark grey, navy, cold blues, or purple.
NEVER USE: Generic grey dashboard aesthetics.
MUST FEEL: Warm, premium, crafted, trustworthy, Bengali artisan.
